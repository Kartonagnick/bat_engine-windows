Bat-Engine - движок, задача которого - автоматизация сборки.  

Движок умеет:  
- искать компиляторы и IDE, которые установленны на компьютере.  
- сценарии сборки на cmake.  
- собирать сразу множество кофигураций.  
- для множества компилятов.  
- запускать тесты.  
- и др.
